html（Hyper Text Markup Langlage）：超文本标记语言（不分大小写）

元素：

```html
<title>百度-下<title>
开始标签 内容  结束标签 
```

主体结构

```html
<!-- 外层：父元素 -->
<html>
    <!-- 内层：子元素 -->
    <head>                     
    </head>                  
    <!-- 内层：子元素 -->
    <body>
    </body>
    <!-- heady、body同级：兄弟元素 -->
</html>
```
