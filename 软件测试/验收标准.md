# 4. 筛选

* 点击后变成“ ^ ” 
* 点击筛选区域弹出下拉框，再次点击应关闭下拉框
* 点击其他筛选区域，应关闭当前下拉框并弹出其他筛选下拉框
* **从哪点击回哪去**

# 5. 弹窗

* 提示语应简洁明了，例如：“长度不符”，不可取
* 一般应为弹窗强提示，极少用toast弱提示
* 弹窗应弹在当前页面
* 异步弹窗
